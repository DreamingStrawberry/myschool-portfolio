## 📋 프로젝트 소개

School Management System은 교육 기관의 관리 업무를 효율화하기 위해 설계된 종합 웹 애플리케이션입니다. 학생과 교사 정보 관리부터 성적 관리, 커뮤니케이션 도구까지 학교 운영에 필요한 모든 기능을 제공합니다.

### 💡 주요 목표
- 교육 기관의 관리 업무 디지털화 및 자동화
- 학생 기록의 정확한 추적 및 관리
- 교사와 관리자를 위한 효율적인 도구 제공
- 교육 커뮤니티 내 원활한 의사소통 지원

## ✨ 주요 기능

### 👥 사용자 관리
```
✅ 역할 기반 접근 제어 (RBAC) - 관리자, 교사별 차등 권한 부여
✅ 안전한 인증 시스템 - 암호화된 비밀번호 저장 및 검증
✅ 사용자 프로필 관리 - 개인 정보 및 설정 커스터마이징
```

### 👨‍🎓 학생 관리
```
✅ 종합적인 학생 정보 관리 - 인적 사항, 연락처, 학업 정보
✅ 반별 학생 조직 - 효율적인 분반 및 학생 그룹 관리
✅ 학생 검색 및 필터링 - 다양한 조건으로 학생 목록 조회
```

### 🔍 주요 테이블 설명

<details>
<summary><b>Student (학생)</b></summary>
<br>

- **id**: 고유 식별자
- **ban**: 소속 반
- **bunho**: 번호
- **name**: 학생 이름
- **phone**: 연락처
- **birth**: 생년월일
- **delstate**: 삭제 상태 (0: 활성, 1: 삭제)

</details>

<details>
<summary><b>Teacher (교사)</b></summary>
<br>

- **id**: 고유 식별자
- **level**: 권한 레벨 (100: 관리자, 80: 담임, 50: 일반 교사)
- **ban**: 담임 반 (있는 경우)
- **userid**: 로그인 아이디
- **pwd**: 암호화된 비밀번호
- **name**: 교사 이름
- **email**: 이메일 주소
- **phone**: 연락처
- **writeday**: 등록일
- **delstate**: 삭제 상태 (0: 활성, 1: 삭제)

</details>

<details>
<summary><b>Sungjuk (성적)</b></summary>
<br>

- **id**: 고유 식별자
- **kor, eng, mat, sci, mus**: 각 과목 점수
- **ban**: 반
- **bunho**: 번호
- **name**: 학생 이름
- **tot**: 총점
- **avg**: 평균

</details>

<details>
<summary><b>Community (게시판)</b></summary>
<br>

- **id**: 고유 식별자
- **title**: 게시글 제목
- **content**: 게시글 내용
- **writeday**: 작성일
- **cla**: 게시판 분류 (notice: 공지사항, board: 자유게시판, data: 자료실)
- **readnum**: 조회수
- **ofname**: 원본 파일명
- **sfname**: 저장 파일명
- **userid**: 작성자 ID

</details>

<details>
<summary><b>Memo (메시지)</b></summary>
<br>

- **id**: 고유 식별자
- **title**: 메시지 제목
- **content**: 메시지 내용
- **fname**: 파일명
- **writeday**: 작성일
- **seUserid**: 발신자 ID
- **reUserid**: 수신자 ID
- **state**: 읽음 상태 (0: 안읽음, 1: 읽음)
- **ofname**: 원본 파일명
- **sfname**: 저장 파일명

</details>

## 🚀 설치 및 실행 방법

### 📋 시스템 요구사항

- JDK 17 이상
- Maven 3.8 이상
- MariaDB 10.5 이상
- 최소 2GB RAM, 10GB 저장 공간

### ⚙️ 설치 과정

<details>
<summary><b>1️⃣ 소스코드 다운로드</b></summary>
<br>

```bash
# 저장소 클론
git clone https://github.com/MeronaKiller/myschool.git

# 프로젝트 디렉토리로 이동
cd myschool
```

</details>

<details>
<summary><b>2️⃣ 데이터베이스 설정</b></summary>
<br>

```sql
-- MariaDB 접속
mysql -u root -p

-- 데이터베이스 생성
CREATE DATABASE school;
USE school;

-- 제공된 스크립트로 테이블 생성 및 초기 데이터 삽입
SOURCE myschool.sql;
```

</details>

<details>
<summary><b>3️⃣ 애플리케이션 설정</b></summary>
<br>

`src/main/resources/application.properties` 파일을 열고 필요한 설정을 수정합니다:

```properties
# 데이터베이스 연결 설정
spring.datasource.url=jdbc:mariadb://localhost:3306/school
spring.datasource.username=root
spring.datasource.password=1234

# 서버 포트 설정
server.port=8089

# 파일 업로드 설정
spring.servlet.multipart.max-file-size=10MB
spring.servlet.multipart.max-request-size=10MB
```

</details>

<details>
<summary><b>4️⃣ 빌드 및 실행</b></summary>
<br>

```bash
# Maven으로 빌드
mvn clean package

# 애플리케이션 실행
java -jar target/school-0.0.1-SNAPSHOT.jar
```

웹 브라우저에서 `http://localhost:8089` 접속

</details>

## 📝 개발 노트

### 주요 구현 포인트
- **역할 기반 접근 제어**: 사용자 권한에 따른 차등적 메뉴 및 기능 접근
